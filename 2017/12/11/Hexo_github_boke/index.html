Hexo + github 搭建个人博客

摘要：
1.先下载安装git和node.js，然后配置hexo(可酌情参考官网文章),2.创建git本地仓库及远程仓库并使之互联，3.hexo文件

1.先得有git和node.js环境，安装hexo之前，最好先配置好git文件夹（创建git本地仓库）

2.然后再git bash中安装hexo.

至此，在本地即可查看博客站点，默认访问地址为：localhost:4000（具体可看运行结果）。
新创建的源文件默认位于博客目录下面的source_posts文件夹下，hexo支持markdown。
只需运行hexo new 文件名 即可新建一篇博客（尚未试验）

3.关联本地博客与远端仓库（即为关联git本地仓库至远程仓库）

  • 创建远程仓库[Github]:https://github.com/

  • 配置SSH key

    成功的话显示为：Hi arron-j! You’ve successfully authenticated, but GitHub does not provide shell access.

    新设密码为打开key的密码（可以不设置直接回车），最终会生成一个文件在用户目录下，打开用户目录，找到.ssh\id_rsa.pub文件，记事本打开并复制里面的内容，打开你的github主页，进入个人设置 -> SSH and GPG keys -> New SSH key：

  • 配置_config.yml中有关deploy的部分

  • 运行hexo

若$ hexo d报错，安装插件$ npm install hexo-deployer-git --save
